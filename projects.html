Vanilla JS Weather App

Built with HTML, CSS, Java Script, Editor, API, GitHub, Bootstrap, Hosting: Netlify.

Created an interactive Vanilla Weather App, that uses the OpenWeatherMap AP1 to give real data and forcast for any location arounfd the world. Find your city by using the search function or clicking the current location icon. It searches for and dispalys the current temperature, humidity, wind, and weather-icon for the searched city. You can also choose to view the temperature in °C or °F.
